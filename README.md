# Quantum-Enhanced Artificial Neural Network for Medical Image Compression

## 📌 Overview

In this project, I worked on building a **Quantum-Enhanced Artificial Neural Network (QANN)** to compress medical images efficiently. The main idea was to reduce the amount of storage needed while still keeping the image quality high enough for medical use. This is important because hospitals and healthcare systems generate a huge amount of imaging data every day, and storing it efficiently is a real challenge.

---

## 🚀 Key Highlights

* Built a **quantum-inspired neural network model** for image compression
* Achieved good compression with **minimal loss in image quality**
* Evaluated the performance using **PSNR (Peak Signal-to-Noise Ratio)**
* Designed the system keeping **real-world medical applications in mind**
* Ensured that reconstructed images retain **diagnostic quality**

---

## 🛠️ Technology Stack

* **Language:** Python
* **Libraries:** NumPy, OpenCV, TensorFlow / PyTorch
* **Tools:** Jupyter Notebook

---

## ⚙️ Approach

The project follows a simple and clear pipeline:

1. First, the medical images are **preprocessed** (cleaned and normalized)
2. Then, important features are extracted using a neural network
3. The images are **compressed using the QANN model**
4. Compressed images are then **reconstructed**
5. Finally, the results are evaluated using **PSNR and similar metrics**

---
